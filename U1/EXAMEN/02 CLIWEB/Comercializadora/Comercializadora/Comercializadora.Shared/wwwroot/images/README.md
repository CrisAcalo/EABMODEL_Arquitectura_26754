# Carpeta para imágenes estáticas

Esta carpeta contiene las imágenes que se usan en la aplicación.

## Para el login background:
1. Coloca tu imagen aquí con el nombre `login-bg.jpg`
2. O cambia el nombre en LoginPage.razor.css

## Formatos soportados:
- .jpg, .jpeg
- .png 
- .gif
- .svg
- .webp

La ruta en CSS será: `url('/images/tu-imagen.jpg')`
